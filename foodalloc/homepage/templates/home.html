FoodAlloc is a food and diet management software that can suggest the user food items based on the user's input. User can select one of four options -

  • Food Allocation - User can enter details and the software will allocate food to the user.
  • Food Lookup - User can lookup information about certain food items.
  • Set Preferences - User can set certain food items as preferences.
  • Set Alternatives - User can set certain food items as alternatives.
